¶Changing state

状态的改变不能直接访问控制，

它会出现错误消息说状态是read-only!的。我们需要使用setState方法来改变状态。

我们仅能在构造函数中初始化State，其它类组件任何地方都不能初始化状态。以及，状态的更改，仅能在该组件内调用setState实现。
